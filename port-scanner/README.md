# Networking Projects

Mini projects exploring networking, security, and Linux.

## Projects

### Port Scanner
A multithreaded TCP port scanner built from scratch in Python.
- Scans a target IP for open ports 1-1024
- Identifies common services by port number
- Uses threading for fast parallel scanning

## Concepts Explored
- TCP socket programming
- Threading in Python
- Network reconnaissance
- How ports and services work
- Traceroute and packet routing
- SSH protocol internals

## Tools Used
- Python3
- Linux terminal
- nmap, mtr, traceroute
